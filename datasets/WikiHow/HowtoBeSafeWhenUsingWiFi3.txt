@article
 Navigate to Start >> Control Panel >> Network and Internet >> Network and Sharing Center. Select “Homegroup and Sharing Options” followed by “Change Advanced Sharing Settings”. From within the Change Advanced Sharing Settings dialog you can disable printer and file sharing as well as Public folder sharing and network discovery.
 Select sharing from the System Preferences window and uncheck all of the sharing options that are currently enabled to disable them.
